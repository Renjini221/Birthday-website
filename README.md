# Birthday-website
Features
Smart Onboarding: simple login screen to personalize the experience with the user's name and birth date.
Dual Modes:
Anticipation Mode: Displays a live countdown timer (Days, Hrs, Mins, Secs) if the birthday hasn't arrived yet.
Celebration Mode: Activates automatically on the birthday! Unlocks the party features.
Interactive Fun:
🎈 Balloons: Floating balloons that you can pop by clicking.
🕯️ Light the Candles: Click to light the candles on the virtual cake with a firework burts.
🎁 Open Gift: A toggleable gift box that reveals a heartwarming hidden message.
🎊 Confetti Cannon: A button to blast confetti all over the screen.
🔮 Crystal Ball: a "Future Prediction" generator for fun fortunes.
🎵 Music Player: Background music support (requires user interaction to play).
Visual Effects:
3D Tilt effect on the main card.
Animated background shapes.
Smooth transitions and typing effects.
🛠️ Built With
HTML5 - For structure and semantic markup.
CSS3 - Custom styling, animations, flexbox layouts, and variables for easy theming.
JavaScript (Vanilla) - Handles all the logic, time calculations, canvas rendering for confetti, and DOM manipulation.
